# 新青年APP - 开源言论平台

新青年APP，适用于安卓、iOS、Web。

新青年，名字来源于二十世纪陈独秀先生的新青年报，旨在创建一个言论自由的平台。

## 言论自由

言论自由，一种基本人权，指公民可以按照个人的意愿表达意见和想法的法定政治权利，
这些意见表达不用受任何人“事前”的审查及限制，也无需担心受到政府或他人报复。
有时也被称为意涵更广泛的表达自由。它通常被理解为充分表达意见的自由，
当中包括以任何方式寻找、接收及发放传递资讯或者思想的行为。
